PURPOSE: To first find the keyword length of the cipher. This is accomplished by calculating the index of coincidence. Next, the step is doing frequency analysis to find the possible keyword for the ciphertext. Finally, once the keyword is found, it is then decrypted into plaintext.

Type the command "python3 IC.py" in your given terminal in order to run this script. Our final answers are in the "results.md" file.
